CIDR Block

 A identidade do ip é representado por um conjunto de quatro número separado por ponto. 
 O valor pode ser definido dentro de conjunto entre 0 e 255.
 Ou seja o intervalo desse campo pode variar entre 0.0.0.0 a 255.255.255.255.
 
 Portanto, o endereço de ip deve possuir 32 bits
 Cada digito --> 1 byte = 8 bits ∴ Total: 32 bits (4 digitos * 8 bits)


  O valor definido a baixo representa um CIDR bloco que utiliza os 
  2 primeiros digitos, (os 16 primeiros bits) para determinar o valor da rede.
 - 10.0.0.0/16
