To install:
    $ curl "https://awscli.amazonaws.com/awscli-exe-linux-x86_64.zip" -o "awscliv2.zip"
    $ unzip awscliv2.zip
    $ sudo ./aws/install
  
 
Login: 
 $ aws configure:
 	AWS Access Key ID: Chave de acesso criado para o usuário.
	AWS Secret Access Key: Chave de acesso criado para o usuário (Secret)
	Default region name [us-east-1]: <region-name>
	Default output format [JSON]: <output-format>
